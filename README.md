# Code Directory

This directory contains all source code and configurations for the CMSL framework.

## Structure

- `camb_cmsl/` - Modified CAMB with k-essence evolution
- `class_cmsl/` - Modified CLASS with O_μν perturbations  
- `mcmc_configs/` - MontePython configuration files
- `likelihoods/` - Cosmological data and likelihood scripts
- `utils/` - Analysis and plotting tools
- `numerical_validation/` - 2D/4D entropy conservation codes
